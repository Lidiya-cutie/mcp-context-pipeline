# Russian PII Support

Пайплайн MCP Context Pipeline полностью поддерживает российские типы персональных данных из конфигурации `personal_data_test_pool.json`.

## Поддерживаемые российские сущности

| Сущность | Плейсхолдер | Пример | Детекция |
|----------|-------------|-------|----------|
| **Паспорт РФ** | `[MASKED_PASSPORT]` | `45 11 123456` | Regex (серия + номер) |
| **ИНН** | `[MASKED_INN]` | `772816897563` | Regex (10 или 12 цифр) |
| **СНИЛС** | `[MASKED_SNILS]` | `112-233-445 95` | Regex (XXX-XXX-XXX XX) |
| **Водительское удостоверение** | `[MASKED_LICENSE]` | `77 11 123456` | Regex (2+2+6 цифр) |
| **Банковский счет** | `[MASKED_ACCOUNT]` | `40817810099910004321` | Regex (40 + 18 цифр) |
| **БИК** | `[MASKED_BIC]` | `044525225` | Regex (04 + 7 цифр) |
| **Автомобильный номер** | `[MASKED_PLATE]` | `А123ВС 77` | Regex (буква + 3 цифры + 2 буквы + 2-3 цифры) |
| **Telegram** | `[MASKED_TELEGRAM]` | `@username` | Regex (@ + username) |
| **VK профиль** | `[MASKED_VK]` | `https://vk.com/id1234567` | Regex (URL) |
| **Полис ОМС** | `[MASKED_POLICY]` | `7755 1234567890` | Regex (4 + 10 цифр) |
| **Клиентский ID** | `[MASKED_CLIENT_ID]` | `CL-00012345` | Regex (CL-XXXXX, USR-XX-XXX-AB, CID_XXXX_XX_XXX) |
| **Номер договора** | `[MASKED_CONTRACT]` | `ДОГ-2026/04-1187` | Regex (ДОГ-YYYY/MM-XXXXX, CNTR-XX-XXXXXX, AG-XX/MM/YYYY-XXXX) |
| **Гео-координаты** | `[MASKED_COORDINATES]` | `55.755826, 37.617300` | Regex (lat, long) |
| **Сложные имена** | `[MASKED_NAME]` | `Иванов-Смирнов Иван Петрович` | Regex (двойные фамилии/имена) |

## Примеры использования

### Базовое использование

```python
from src.pii_guard import get_pii_guard

guard = get_pii_guard()

text = """
Меня зовут Иванов Иван Иванович, паспорт 45 11 123456,
ИНН 772816897563, телефон +7 (999) 123-45-67,
email: ivan.ivanov@example.com.
"""

masked = guard.mask(text, language='en')

print(masked)
# Output:
# Меня зовут [MASKED_NAME], паспорт [MASKED_PASSPORT],
# ИНН [MASKED_INN], телефон [MASKED_PHONE],
# email: [MASKED_EMAIL].
```

## Интеграция с MCP сервером

PII Guard автоматически интегрирован в MCP сервер:

```python
# В server.py
from pii_guard import get_pii_guard

@mcp.tool()
async def compress_context(messages: List[str], session_id: str) -> dict:
    """
    Безопасная суммаризация с автоматическим маскированием PII.
    """
    text_block = "\n\n".join(messages)

    # Автоматическое маскирование PII
    masked_text = get_pii_guard().mask(text_block, language='en')

    # Отправка маскированного текста в LLM
    summary = await call_llm_for_summary(masked_text, system_message)

    # Сохранение результатов
    # ...
```

## Приоритизация сущностей

Для обработки перекрывающихся сущностей используется система приоритетов:

1. **Высокий приоритет** (специфичные российские сущности):
   - Bank Account, SNILS, INN, Passport RF
   - Driver License, Medical Policy, BIC Code

2. **Средний приоритет** (финансовые данные):
   - Credit Card, Vehicle Plate
   - Telegram, VK Profile, Client ID

3. **Низкий приоритет** (общие сущности):
   - Email, Phone, Person, Location, Date

Это гарантирует, что `40817810099910004321` будет замаскирован как `[MASKED_ACCOUNT]`, а не как `[MASKED_PHONE]`.

## Ограничения

1. **Имена на русском языке**: Частично детектятся через паттерны
   - Сложные имена с дефисами (Иванов-Смирнов, Анна-Мария) детектятся полностью
   - Простые русские имена могут не детектиться NER (Presidio поддерживает только английский)
   - Решение: Используйте контекст или добавляйте известные имена в deny_list

2. **Адреса**: Только обнаружение отдельных локаций (города, улицы)
   - Полные адреса требуют более сложного NER

3. **Частичные имена**: Инициалы и сокращения могут не детектиться
   - Дополнительные паттерны могут быть добавлены по необходимости

## Поддерживаемые форматы телефонов

Пайплайн поддерживает все основные форматы российских телефонных номеров:

- `+7 (999) 123-45-67` - стандартный формат с кодом страны
- `+7 999 123 45 67` - с пробелами
- `8 (999) 123-45-67` - формат с 8 вместо +7
- `89991234567` - без разделителей
- `+7-921-555-44-33` - с дефисами
- `+7(905)7008090` - без пробелов после скобок
- `+79265005566` - без разделителей
- `8 800 555-35-35` - городской формат

Все форматы детектируются и маскируются как `[MASKED_PHONE]`.

## Поддерживаемые сложные имена

Пайплайн поддерживает сложные русские имена с дефисами:

- `Иванов-Смирнов Иван Петрович` - двойная фамилия
- `Анна-Мария Сидорова Викторовна` - двойное имя
- `Иванов-Петров Иван-Алексей Михайлович` - двойная фамилия и двойное имя
- `Павлова Екатерина Дмитриевна-Мартиросян` - двойное отчество

Все сложные имена детектируются и маскируются как `[MASKED_NAME]`.
